# Coffee Shop Custom Code

Custom application code from my Laravel coffee shop project.

## Included
- products
- product attributes
- select options for attributes
- cart with grind type
- checkout and orders
- fulfillment statuses
- admin panel
- customer management

## Stack
- PHP
- Laravel
- Blade
- MySQL

## Notes
This repository contains project-specific application code only.
Framework core, vendor dependencies, environment files, and build artifacts are not included.
